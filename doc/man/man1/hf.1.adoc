== NAME

hf - an utility for hidden file/directory

== SYNOPSIS

*{manname}* [_OPTION_]... _FILE_...

== DESCRIPTION

*{manname}* is a command-line utility for hidden file/directory.
By default, *{manname}* will show what would be done when you hide files or
directories without make any changes (same as when both *--hidden* and
*--dry-run* are specified).
